Como rodar?
1: git clone https://github.com/pitocoofc/NexosDetecte.git
2: cd NexosDetecte
3: cd 1.0
4: mv rodar.js Pesos
5: cd Pesos
6: nano texto.txt
7: Adicione seu texto
8: node rodar.js
