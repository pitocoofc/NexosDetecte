Como rodar?
1: git clone https://github.com/pitocoofc/NexosDetecte.git
2: cd NexosDetecte
3: cd 1.0
4: mv rodar.js Pesos
5: cd Pesos
6: nano peso.json
7: Renomeie ele de "modelo_ia.json"
8: nano texto.txt
9: Adicione seu texto
10: node rodar.js
